PROPOSITO

El proposito de este sitio web es fomentar el aprendizaje de diseño de sitios web enfocados en el uso de Frameworks CSS, Posicionamiento Web

Frameworks CSS

Definicion

Un framework de CSS, es una biblioteca de estilos visuales genericos que puede ser usada para implementar diseños web,lo cual aportan una serie de utilidades que pueden ser aprovechadas frecuentemente en los distintos diselos web, tales como componentes, cuadros de dialogo, tablas, carrucel de imagenes entre otros, lo cual los mas usados son, uikit framework, Foundation, Bulma, Semantic UI.

VENTAJAS DE USAR UN FRAMEWORK CSS

  • Acelebra el desarrollo de tus paginas.
  • Facilita el trabajo en equipo.
  • Garantiza la compatividad entre los distintos navegadores.
  • Son mantenidos y mejorados por la comunidad.

DESVENTAJAS

  • Hace la carga de tus paginas mas lenta añade cosas a tu proyecto que no vas a usar.
  • Te lleva a diseñar a diseñar paginas muy similares a todas las demas hechas con este framework.
  • Tienes que aprender a usarlos y ingnorar los fundamentos de CSS.

Posicionamiento Web

Definicion

El posicionamiento en buscadores, optimización en motores de búsqueda o SEO, es un conjunto de acciones orientadas a mejorar el posicionamiento de un sitio web en la lista de resultados de Google, Bing, u otros buscadores de internet.

VENTAJAS DE USAR UN POSECIONAMIENTO WEB

  • Alta rentabilidad a largo plazo.
  • Influye los contenidos de la web su actualización y su estructura.
  • Solo honorarios SEO.
  • Los usuarios hace mas facilmente click en los resultados organicos que en los patrocinados.

DESVENTAJAS DE USAR UN POSECIONAMIENTO WEB

  • Resultados a medio o largo plazo.
  • Solo es posible posicionar un reducido numero de palabras claves.
  • Google elige la landing page.
  • No podemos determinar el mensaje.
  • No se pueden predecir ni garantizar resultados.
  • No es posible segmentar.
